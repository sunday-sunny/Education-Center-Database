![2  전체ERD](https://user-images.githubusercontent.com/42857790/145359778-b694b7ea-7d5e-4977-b894-fd9b4376b669.PNG)

**👩‍🏫 교육센터 운영 프로그램 데이터베이스 구축 프로젝트입니다.**
- 교육기관에서 **과정을 구성**하고 **강의 일정을 운영**하는 것을 목표로 합니다.
- **관리자/교사/교육생** 계정 별 요구사항에 맞는 **뷰와 맞춤 데이터** 제공합니다.
- **취업 지원, 통계 기능**을 제공하여 교육생 관리를 용이하게 합니다.
<br/>

### 👑 Detail
### 1. 강의스케줄 조회
![image](https://user-images.githubusercontent.com/42857790/145665001-70ca1774-ec6b-40d3-afff-e9dc6c45d2af.png)
#### 1-1. 교사별 강의스케줄 확인
- 교사는 자신의 강의 스케줄을 확인할 수 있습니다.
- 강의 스케줄은 강의예정/강의중/강의종료로 구분해서 확인 가능합니다.
<br/>
<br/>
<br/>


![image](https://user-images.githubusercontent.com/42857790/145665960-5a1e2a26-d385-44df-bde5-121a8154f7c4.png)
#### 1-2. 특정 과목 교육생 정보 출력
- 특정 과목의 과목번호 선택시 해당 과정에 등록된 교육생 정보를 확인할 수 있습니다.
<br/>
<br/>


### 2. 배점 입출력
![image](https://user-images.githubusercontent.com/42857790/145665539-2d316d26-c48b-4932-859b-c193383a3693.png)
#### 2-1. 교사별 '강의종료' 과목 정보 출력
- 교사는 강의를 마친 과목에 대한 성적 처리를 위해 배점 입출력을 할 수 있습니다.
- 배점 입출력을 위한 강의종료 과목 정보와 출결, 필기, 실기 배점 정보, 시험문제등록여부를 출력합니다.
<br/>
<br/>
<br/>


![image](https://user-images.githubusercontent.com/42857790/145665681-dea49509-fbc6-4ab4-bd80-20851cb83535.png)
#### 2-2. 특정 과목 배점 및 시험날짜/문제 입력 정보 출력
- 특정 과목의 과목번호 선택시 '출결 배점, 필기 배점, 실기 배점, 시험 날짜, 시험 문제'를 입력할 수 있는 화면을 보여줍니다.
<br/>
<br/>


### 3. 성적 입출력
![image](https://user-images.githubusercontent.com/42857790/145665831-fb9283d6-d463-4dd2-9ed4-90f6f66609d7.png)
#### 3-1. 교사별 '강의종료' 과목 정보 출력
- 교사는 강의를 마친 과목에 대한 성적 처리를 위해 성적 입출력을 할 수 있습니다.
- 배점 입출력을 위한 강의종료 과목 정보와 출결, 필기, 실기 배점 정보, 성적등록여부를 출력합니다.
<br/>
<br/>
<br/>


![image](https://user-images.githubusercontent.com/42857790/145665936-79e872e7-8ce9-464d-8210-763f4aa1f9fc.png)
#### 3-2. 특정 과목 교육생 정보 및 성적 출력
- 특정 과목의 과목번호 선택시 교육생 정보 및 성적을 출결, 필기, 실기 총점으로 구분하여 보여줍니다.
- 중도 탈락하여 성적 처리가 제외된 교육생이더라도 교육생 명단에 출력됩니다.
- 중도 탈락생은 '중도 탈락 여부'와 '중도 탈락 날짜'를 함께 출력합니다. 
<br/>
<br/>
<br/>


### 4. 출결 관리 및 출결 조회
![image](https://user-images.githubusercontent.com/42857790/145666151-399a69fa-d45c-4113-bbdf-9ac53c4793e5.png)
- 교사가 강의한 과정에 한해 선택하는 경우 모든 교육생의 출결을 조회할 수 있습니다.
- 출결 현황을 기간별(년, 월, 일), 특정 과정별 조회 가능합니다.
- 모든 출결 조회는 근태 상황을 구분할 수 있습니다. (정상, 지각, 조퇴, 외출, 병가, 기타)

## 🍰 후기
자바 프로젝트에 이어 데이터베이스 프로젝트도 마무리되었습니다. 역시나 3년 전 데이터베이스 수업 이후로 참 오랜만이었는데요.😅
<br/>

당시와 다르게 아무래도 팀 프로젝트이다 보니 규모도 커지고 SQL을 짜는 범위도 늘어났습니다. DBMS를 쓸 줄도 모르던 꼬꼬마 시절 때는 콘솔로 mariaDB를 켜놓고 한땀한땀 데이터를 넣는 짓을 했었는데.. 여러모로 패기 있고 삽질 했던 시절입니다.. 지금도 가끔 마주치는 조교님께 프로그램을 쓸 수 있는 인간이 되었다고 이 기쁜 소식을 알려드려야겠습니다.😎
<br/>

인간이 도구를 쓰며 진화해왔듯, 저라는 인간도 이번 팀 프로젝트를 통해 여러가지 프로그램과 기술을 도입해 볼 수 있었습니다. DBMS와 ERD 프로그램을 사용해 볼 수 있었고 꼭 써보고 싶었던 클라우드 서버도 쓸 수 있어 좋았습니다. 항상 느끼지만 효율적인 협업을 위한 클라우드는 필수불가결한 요소라 생각합니다.
<br/>

저의 농담을 받아준 팀원들에게도 무한한 감사를 보냅니다!💗
