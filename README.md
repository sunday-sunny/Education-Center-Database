
# 👩‍🏫 Education Center Database
![2  전체ERD](https://user-images.githubusercontent.com/42857790/145359778-b694b7ea-7d5e-4977-b894-fd9b4376b669.PNG)
<br/>

###  👩‍🏫 교육센터 운영 프로그램 데이터베이스 구축 프로젝트입니다.
- 교육기관에서 **과정을 구성**하고 **강의 일정을 운영**하는 것을 목표로 합니다.
- **관리자/교사/교육생** 계정 별 요구사항에 맞는 **뷰와 맞춤 데이터** 제공합니다.
- **취업 지원, 통계 기능**을 제공하여 교육생 관리를 용이하게 합니다.
<br/> 

### Source Directory
```
┌─1. 요구분석서
├─2. ERD
├─3. ⭐테이블 정의서		> DDL
├─4. ⭐데이터 스크립트
│  └─ 데이터SQL			> DML(DATA INSERT File)
├─5. ⭐업무 스크립트
│  ├─ PL-SQL			> Procedure / Function
│  └─ 업무SQL			> DML(ANSI-SQL)
├─6. 데이터
├─7. PPT
├─8. 최종요약본
└─⭐개인src			> 담당업무
```
